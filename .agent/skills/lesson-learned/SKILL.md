# 技能：记录教训 (Lesson Learned)

## 目标
将这次错误提炼为一条结构化记忆，写入对应的 MEMORY.md，让 Antigravity 下次不再重犯。

## 判断写入哪个文件

- 错误与**某个项目的技术实现**有关 → 写入 `projects/{project_name}/MEMORY.md`
- 错误与**工作流程或沟通方式**有关 → 写入根目录 `MEMORY.md`

## 记录格式

在对应 MEMORY.md 中追加以下格式的条目：

```markdown
## [{YYYYMMDD}] {一句话描述错误}

**错误行为**：我做了什么，导致了问题。
**正确做法**：应该怎么做。
**触发场景**：什么情况下容易再犯（用于下次启动时的自我检查）。
```

## 执行步骤

1. 检查对应 MEMORY.md 是否存在，不存在则创建（写入标题 `# Antigravity 记忆文件`）
2. 提炼本次错误，按格式写入新条目
3. 向用户确认记录内容，告知写入了哪个文件
